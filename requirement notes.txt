Steps to Run this project 
------------------------------------------------------------
1. Download Intellij
2. open project, make sure folder has a small black box
3. click File->project structure->modules Tab->dependencies
4. click plus sign->add library->java->new library->locate the libs folder->ok
5. add user & pass for aggie access

note: If these dont work that means you will have to download:

Selenium Client & WebDriver Language Bindings 
-https://www.selenium.dev/downloads/

ChromeDriver
-https://chromedriver.chromium.org/downloads

ps. 
Also works with other IDEs, just import the jar files in libs folder
